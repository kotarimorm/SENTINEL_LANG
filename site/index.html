Overview

Sentinel Lang is an experimental systems programming language designed for low-level development. It compiles Sentinel source code into NASM-style assembly and then into a flat binary.

The language is designed as a practical middle ground between readable high-level syntax and direct low-level control.

Main idea

    (3) vga_print("send eoi done")

lib(std)

First built-in OSDev helper layer.

Status

In v0.5-alpha, lib(std) remains the first Sentinel OSDev helper pack. It is currently x64-only and was stress-tested through the Kernel Beast validation path.

Commands

Command Purpose
vga_print(text) Print text to VGA buffer.
vga_clear() Clear VGA text buffer.
halt() Emit safe halt loop.
nop() Emit NOP.
io_wait() Emit small I/O wait.
write_port(port, value) Write byte to I/O port.
read_port(port) Read byte from I/O port.
irq_disable() Emit CLI.
irq_enable() Emit STI.

x64-only rule

lib(std)
x64
